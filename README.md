# Hello Claude

A tiny practice project for shipping my first pull request with Claude Code.

## Usage

Run the greeter:

    python greet.py

## Example

Running the greeter prints a friendly message:

    $ python greet.py
    Hello, world!

You can also import and call it directly:

    from greet import greet

    greet()            # "Hello, world!"
    greet("Wilson")    # "Hello, Wilson!"

## About

This repo was created to learn the GitHub pull request workflow:
branch, make a change, commit, push, and open a PR.
